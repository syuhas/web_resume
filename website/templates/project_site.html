Website Build

Overview

This site was built not only to start compiling completed projects into one place, but also because the site is a project in and of itself. It has been built from the ground up using AWS for features like hosting, CI/CD aspects, DNS services, load balancing, caching and TLS authentication in addition to coding all the HTML and CSS via a Python Flask app which is lightweight and gives the freedom to add features in the future as desired.

Python Flask / HTML / CSS

Starting with a simple   Python Flask   app, the necessary groundwork needed for the basic outline of the site can be built. This includes the blueprint that will link to all of the html templates to each route on the base html template.

Now the base   HTML   template can be built that will act as the basic design of the site, including the navigation bar, footer and background design.

Using   Bootstrap   for a navigation bar we now have a working skeleton to add pages to. Jinja is used for the templating in Flask so we can extend the additional html stemming from this base page.

Using an external   CSS   stylesheet, the look of the site can be completely controlled in one place. A combination of   Grid   template areas and   FlexBox   is used so the layout of the site can be fine tuned and allow for media queries for viewing on different devices. This is where the bulk of the design for the site takes place.

All of these components combined make up the whole of the site. In addition to using all of these elements, the site was created using a   Virtual Environment   so only the necessary   Dependencies   are kept locally in one place. This makes things easier when creating and uploading a   requirements.txt   file to be used with Elastic Beanstalk in the future, and ensures that no unecessary dependencies are installed.
